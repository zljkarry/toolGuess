# 五一劳动器具猜猜看

## 项目简介

五一劳动工具猜猜看小游戏，分简单、一般、困难，分别计算答题速度和排名，只有前两次闯关记录成绩。有强制了解器具的时间限制。

产品原型图：https://lanhuapp.com/url/1cZ5I

## 项目名

toolGuess
